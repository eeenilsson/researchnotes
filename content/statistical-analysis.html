3.1 Basics

You should acquaint yourself with some common statistical terms.

Statistics are used to describe and summarise the data (descriptive statistics) and to calculate measures of association between variables. Some basic statistical terms that you should be familiar with are listed below.

  • Normal distribution. Whether the data is normally distributed decides what statistics are appropriate to describe it and to test associations.

3.2 Hypothesis testing

In contrast to descriptive statistics, methods used for hypothesis testing provides an estimate or test for differences between groups or associations between variables. The statistical terms below are useful to know.

  • T-test
  • Chi-2-test
  • P-value
  • Power
  • Type I error
  • Type II error
